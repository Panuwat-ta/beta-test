# 🚀 เริ่มต้นใช้งาน Data Links

## ⚡ Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start server
npm start

# 3. Open browser
http://localhost:3000
```

## ✨ ฟีเจอร์ใหม่ที่ต้องลอง!

### 1. 🎨 Dark/Light Theme
- กดปุ่ม 🌙/☀️ ใน Navbar (มุมขวาบน)
- ธีมจะถูกบันทึกอัตโนมัติ

### 2. 🔲 Grid/List View (หน้า Home)
- กดปุ่ม 🔲 = Grid View (แสดงเป็นการ์ด)
- กดปุ่ม ☰ = List View (แสดงเป็นตาราง)
- View ที่เลือกจะถูกบันทึกอัตโนมัติ

### 3. 🔍 Search & Filter
- พิมพ์ในช่องค้นหาเพื่อค้นหา Links
- เลือก Category จาก Dropdown
- ใช้ร่วมกันได้

### 4. 📱 Mobile Responsive
- เปิดบนมือถือได้สบาย
- กดปุ่ม ☰ เพื่อเปิด Menu

### 5. 👤 User Profile
- Login แล้วจะเห็นรูปและชื่อใน Navbar
- กดรูปโปรไฟล์เพื่อดูขนาดใหญ่

## 📱 Navbar ใหม่

```
[🔗 Data Links]  [Home] [Files] [Update] [Upload] [Notes] [Add Note] [About] [Support]  [👤 Username] [🌙]
```

- **ซ้าย**: Brand Logo และชื่อ
- **กลาง**: Navigation Links
- **ขวา**: User Profile และ Theme Toggle

## 📄 หน้าต่างๆ

1. **Home** (`/`) - แสดง Links ทั้งหมด (Grid/List View)
2. **Files** (`/files.html`) - Browse Google Drive Files
3. **Update** (`/date.html`) - เพิ่ม/แก้ไข Links
4. **Upload** (`/upload.html`) - Upload Files to Google Drive
5. **Notes** (`/note.html`) - ดู Notes ทั้งหมด
6. **Add Note** (`/add-note.html`) - เพิ่ม Note ใหม่
7. **About** (`/about.html`) - เกี่ยวกับผู้พัฒนา
8. **Support** (`/suport.html`) - สนับสนุนผู้พัฒนา
9. **Login** (`/login.html`) - Login/Register
10. **Profile** (`/logout.html`) - ดูและแก้ไขโปรไฟล์

## 💡 Tips

1. **Theme**: ลองทั้ง Dark และ Light Theme
2. **View**: Grid View สำหรับดูภาพ, List View สำหรับดูรายละเอียด
3. **Search**: พิมพ์ชื่อ Link หรือ Username เพื่อค้นหา
4. **Mobile**: แอปทำงานได้ดีบนมือถือ
5. **Profile**: Login เพื่อดูโปรไฟล์และจัดการ Links
